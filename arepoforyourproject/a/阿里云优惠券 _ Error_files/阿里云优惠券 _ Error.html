阿里云优惠券

发表于 2018-03-24 | 阅读次数: 73

领取

你可以在这领取→https://promotion.aliyun.com


或者你可以扫二维码↓

使用

以我选购阿里云的ECS为例

  1. 进入阿里云首页->产品->云服务器ECS->立即购买
  2. 选择配置,我这里选择了一键购买，当然你也可以选择自定义购买
  3. 选择完配置之后,选择立即购买,在优惠券那栏选择刚刚领取的优惠券
  4. 最后就是下单，支付就OK了,其他产品也是类似。
